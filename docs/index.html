今回の回答では、生成AIを 「知っている（人に説明できる）」層が 60.9% を占めており、利用頻度では ほぼ毎日使っている が中心です。あわせて、日常的に使っている層は 60.9% まで広がっており、社内でのAI活用は着実に浸透してきています。

一方で、活用を進めるうえでの主なハードルは 情報漏洩・セキュリティが不安（47.8%）、期待した答えが返ってこない（34.8%） で、会社に求められている支援は 社内向けのルール・ガイドライン整備（73.9%）、社内で活用事例を共有する仕組み（69.6%） が中心です。任せ方としては 下書き・アイデア出しまで。最終判断は人がやる が最も多く、全面自動化よりも 人が最終判断する前提で使いやすくする環境整備 への期待が強い状況です。

業務負荷が大きい領域としては 情報収集・リサーチ（56.5%）、文章作成（メール・報告書・提案書 など）（56.5%）、資料・スライド作成（47.8%） が上位にあり、社内セミナーに対しては 他社事例・活用アイデア（73.9%）、注意点・リスク・セキュリティ（73.9%） への関心が高くなっています。全体として、具体的な使い方を共有しながら安全に活用範囲を広げたい という傾向が読み取れます。
